# Simple Calculator

This is a simple calculator web application developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division through an interactive user interface.

## Features

- Addition, subtraction, multiplication, and division operations
- Clear display functionality
- User-friendly interface
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Objective

The objective of this project is to understand JavaScript DOM manipulation, event handling, and basic frontend development concepts.
